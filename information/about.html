About This Website

This website was created on April 28th, 2025 as a fun project for me (T7) to learn about HTML, CSS and JS.


The website contains secrets hidden in each page, sometimes even outside of the website.
Secrets are classified into 5 levels:

  • Level 1
    It is pretty obvious, sometimes hidden in plain sight
  • Level 2
    It requires an ID, which usually appears as a clue in a specific place
  • Level 3
    It requires either a specific ID or action, which is usually hidden in the Console
  • Level 4
    It can only be found in the source code, or with a very specific sequence of actions or inputs
  • Level 5
    You have to be a very talented hacker or a genius to figure this out
